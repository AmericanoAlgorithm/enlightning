OSX 10.10 general install notes (NOT EXACT):

Install gcc (4.9.2) and gfortran (4.9.2) using Homebrew.
    brew install gcc
    brew install gfortran

Install hdf5 (1.8.14) and boost (1.57.0) using Homebrew.
    brew install homebrew/science/hdf5 --with-mpi
    brew install boost --with-mpi

Install openmpi (1.8.4) using Homebrew.
    brew install openmpi

Install SAMRAI.
    mkdir ~/SAMRAI-v3.9.1
    mv ~/Downloads/SAMRAI-v3.9.1.tar.gz ~/SAMRAI-v3.9.1/
    cd ~/SAMRAI-v3.9.1/
    tar xf SAMRAI-v3.9.1.tar.gz
    mkdir build
    cd build
    ../SAMRAI/configure --prefix=~/SAMRAI-v3.9.1/install --with-hdf5=/usr/local --with-CC=mpicc --with-CXX=mpicxx --with-F77=mpif77 --with-parallel-run-bin=mpirun --with-zlib --without-petsc --enable-opt=-O3 --disable-debug --with-boost=/usr/local
    make library -j8

Fix an error in SAMRAI's Fortran location.
    Change the line "INCLUDE_SAM   = $(SAMRAI)/source" in ~/SAMRAI-v3.9.1/install/config/Makefile.config to be "INCLUDE_SAM   = $(SAMRAI)/include".

Install Enlightning.
    Change the SAMRAI variable in the Enlightning makefile to the location of SAMRAI source, i.e. "~/SAMRAI-v3.9.1/install"
    make
